- 높이: 720

적용 방법:
1. 압축을 풀기
2. src 폴더 안 파일들을 기존 프로젝트 src 폴더에 덮어쓰기
3. 새 파일 RandomOverlay.jsx도 src 안에 추가하기
4. npm run dev 로 로컬 확인
5. git add . && git commit -m "Add random overlay route" && git push
6. Vercel 자동배포 확인
